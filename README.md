```bash
#add file to next commit
git add <file>
#remove file from next commit
git rm --cached <file>
#rename file in the next commit
git mv <file>
```

## Stash Changes
1. stash changes
   - ```git stash```
2. list changes
   - ```git stash list```
3. show the contents
   - ```git stash show stash@{0}```
4. apply the last stash
   - ```git stash apply```
5. apply a specific stash
   - ```git stash apply stash@{0}```
